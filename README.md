# WDD 130 Starting Template Repository
This repository is a starting template for students in WDD 130.

## GitHub Pages Configuration

To enable GitHub Pages for this repository, follow these steps:

1. Select the **Settings** tab for the repository.
2. On the left hand menu, select **Pages**.
3. Configure the pages to build from the root of the main branch by doing the following:
   - Leave the **Source** as: **Deploy from a branch**
   - In the **Branch** section, select **main**.
   - After selecting main, leave the location as **/ (root)**, which is the default.
   - Click the **Save** button.

Once configured, your site will be available at: `https://<your-username>.github.io/wdd130/`
